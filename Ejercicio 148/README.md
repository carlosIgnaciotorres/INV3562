# Problema

App de lingüística para análisis de textos en diferentes idiomas.

## Restricciones

- Contar solo vocales (a, e, i, o, u), ignorando acentos.
- Si la palabra está vacía, mostrar 'Error'.
- Diferenciar mayúsculas y minúsculas.
