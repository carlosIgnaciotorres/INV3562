# Problema

Sistema de nómina para calcular salario neto de empleados.

## Restricciones

- Horas trabajadas no pueden ser negativas.
- Valor por hora debe ser ≥ al salario mínimo legal.
- Descuentos sociales no pueden exceder el 15% del total.
