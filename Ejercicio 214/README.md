# Problema

Plataforma de viajes para recordatorios de cumpleaños.

## Restricciones

- Fecha actual debe ser ≤ fecha de cumpleaños en el año.
- Si ya pasó, calcular para el próximo año.
- Mostrar días restantes en verde.
