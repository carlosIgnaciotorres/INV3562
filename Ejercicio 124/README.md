# Problema

Herramienta de reportes empresariales para formatear fechas.

## Restricciones

- Mes debe estar entre 1 y 12.
- Si el mes es inválido, mostrar 'Error'.
- Nombres de meses en minúsculas (ej: 'diciembre').
