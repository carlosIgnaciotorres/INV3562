# Problema

App de matemáticas para verificar propiedades numéricas en tiempo real.

## Restricciones

- Número debe ser ≥ 0.
- No usar funciones de raíz cuadrada.
- Mostrar 'Cuadrado perfecto' o 'No cuadrado perfecto'.
