# Problema

Sistema de validación de tarjetas de crédito para comercios electrónicos.

## Restricciones

- Ignorar caracteres no numéricos.
- Mostrar cantidad de dígitos y suma total.
- Si el número es negativo, mostrar 'Inválido'.
