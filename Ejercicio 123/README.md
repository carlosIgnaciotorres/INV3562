# Problema

Software de planificación de eventos para ajustar fechas clave.

## Restricciones

- Días a sumar ≥ 0.
- Ajustar mes y año si es necesario (ej: 32 días → mes siguiente).
- Mostrar fecha resultante en formato DD/MM/AAAA.
