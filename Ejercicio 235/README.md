# Problema

Herramienta de psicología para estudios de números felices en terapias.

## Restricciones

- Número debe ser ≥ 1.
- Mostrar pasos intermedios si el usuario lo solicita.
- Limitar a 100 iteraciones para evitar bucles infinitos.
