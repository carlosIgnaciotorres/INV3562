# Problema

Sistema de codificación para dispositivos embebidos (ej: microcontroladores).

## Restricciones

- Números enteros entre 0 y 255 (8 bits).
- Mostrar binario con 8 dígitos (ej: 00001010).
- Si el número es negativo, usar complemento a 2.
