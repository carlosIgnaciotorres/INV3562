# Problema

Sistema de gestión de usuarios para plataformas educativas.

## Restricciones

- Formato de entrada: 'Nombre Apellido1 Apellido2'.
- Si no hay apellidos, mostrar 'Nombre: [nombre]'.
- Ignorar nombres vacíos.
