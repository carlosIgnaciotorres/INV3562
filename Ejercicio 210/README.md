# Problema

App de gestión de proyectos para comparar fechas de entrega.

## Restricciones

- Formato de fecha: DD/MM/AAAA.
- Si las fechas son iguales, mostrar 'Misma fecha'.
- Validar que las fechas no sean anteriores a 2000.
