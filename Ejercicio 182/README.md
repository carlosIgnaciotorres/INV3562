# Problema

App de matemáticas recreativas para detectar números de Armstrong.

## Restricciones

- Número debe ser ≥ 0.
- Mostrar descomposición de dígitos elevados a su longitud.
- Si es inválido (ej: negativo), mostrar 'Error'.
