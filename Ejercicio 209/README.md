# Problema

App de gestión de proyectos para calcular antigüedad de tareas.

## Restricciones

- Fecha en formato DD/MM/AAAA.
- Si la fecha es futura, mostrar 'Aún no ocurre'.
- Resultado en días (entero positivo).
