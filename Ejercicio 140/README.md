# Problema

App educativa para enseñar conceptos matemáticos básicos.

## Restricciones

- Clasificar el número en positivo/negativo/cero.
- Si es cero, mostrar ejemplos prácticos (ej: temperatura).
- Validar entradas no numéricas.
