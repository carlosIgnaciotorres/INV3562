# Problema

Sistema de seguridad para verificar números primos gemelos en criptografía.

## Restricciones

- Números deben ser primos y diferir en 2 (ej: 11 y 13).
- Si no son gemelos, mostrar 'No aplica'.
- Limitar la entrada a números ≤ 10^5.
