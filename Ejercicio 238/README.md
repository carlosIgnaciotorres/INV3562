# Problema

App de matemáticas para invertir dígitos en problemas educativos.

## Restricciones

- Manejar números negativos (ej: -123 → -321).
- Si el resultado excede el límite de enteros, mostrar 'Desbordamiento'.
- Ignorar ceros a la izquierda.
