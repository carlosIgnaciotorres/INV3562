# Problema

Software de punto de venta para pequeñas tiendas minoristas.

## Restricciones

- Manejar hasta 100 productos simultáneos.
- Si el total es $0, mostrar 'Carrito vacío'.
- Aplicar impuestos locales automáticamente.
