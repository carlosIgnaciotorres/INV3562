# Problema

Herramienta de marketing para validar correos de campañas masivas.

## Restricciones

- Debe contener '@' y un dominio válido (ej: .com).
- Si no cumple, mostrar 'Correo inválido'.
- Ignorar mayúsculas/minúsculas.
