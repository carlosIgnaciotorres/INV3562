# Problema

App de seguridad para validar números palindrómicos en tarjetas RFID.

## Restricciones

- Ignorar caracteres no numéricos.
- Si es palíndromo, mostrar 'Acceso concedido'.
- Manejar números negativos (ej: -121 → 121).
