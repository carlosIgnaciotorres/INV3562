# Problema

App de matemáticas para enseñar sumatorias a estudiantes de primaria.

## Restricciones

- N debe ser ≥ 1.
- Si N es negativo, mostrar 'Inválido'.
- Mostrar fórmula: 1 + 2 + ... + N.
