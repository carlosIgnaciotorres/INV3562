# Problema

Sistema de seguridad para validar códigos de acceso.

## Restricciones

- Número debe ser divisible por 3 y 7 simultáneamente.
- Si no cumple, mostrar 'Acceso denegado'.
- Bloquear después de 3 intentos fallidos.
