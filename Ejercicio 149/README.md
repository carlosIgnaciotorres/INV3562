# Problema

Plataforma de educación en línea para evaluaciones automatizadas.

## Restricciones

- Calificaciones entre 0 y 100.
- Si el promedio es ≥ 70, mostrar 'Aprobado' en verde.
- Validar que todas las notas sean numéricas.
