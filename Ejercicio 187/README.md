# Problema

Herramienta meteorológica para conversiones de temperatura en laboratorios.

## Restricciones

- Kelvin debe ser ≥ 0.
- Mostrar Celsius y Fahrenheit con 2 decimales.
- Si Kelvin < 0, mostrar 'Inválido'.
