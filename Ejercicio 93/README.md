# Problema

Sistema de pedidos para restaurantes con opciones de menú interactivo.

## Restricciones

- Validar que la opción esté en [1-5].
- Si se ingresa una opción inválida, reiniciar el menú.
- Manejar hasta 3 reintentos antes de cerrar.
