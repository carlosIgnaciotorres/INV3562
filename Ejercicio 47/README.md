# Problema

Herramienta para cartografía digital de terrenos montañosos.

## Restricciones

- Los catetos deben ser ≥ 0.01.
- Si los valores son iguales, indicar 'Triángulo isósceles'.
- Usar el teorema de Pitágoras exacto (sin aproximaciones).
