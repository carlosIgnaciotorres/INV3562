# Problema

Herramienta de inversiones para calcular rendimientos financieros.

## Restricciones

- Intereses deben ser ≥ 0.
- Si intereses ≤ 7000, mostrar 'No reinvertir'.
- Mostrar saldo final con formato $X,XXX.XX.
