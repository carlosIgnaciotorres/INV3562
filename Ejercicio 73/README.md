# Problema

Plataforma de cursos en línea para aprobar certificaciones técnicas.

## Restricciones

- Notas deben estar entre 0 y 10.
- Si el promedio es 7.5, mostrar 'Aprobado (exacto)'.
- Ignorar notas fuera del rango válido.
