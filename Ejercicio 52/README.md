# Problema

Traductor numérico para facturas legales en sistemas contables.

## Restricciones

- Solo números del 0 al 9.
- Si el número está fuera de rango, mostrar 'Inválido'.
- Usar nombres en minúsculas (ej: 'cinco', no 'Cinco').
