# Problema

Plataforma de inversiones para calcular rendimientos ponderados.

## Restricciones

- Valores y pesos deben tener la misma cantidad de elementos.
- Si algún peso es 0, ignorar ese valor.
- Mostrar resultado con 3 decimales.
