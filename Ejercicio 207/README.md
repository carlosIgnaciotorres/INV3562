# Problema

Software de organización de bibliotecas para ordenar títulos alfabéticamente.

## Restricciones

- Ignorar artículos iniciales (ej: 'El', 'La').
- Mostrar palabras en minúsculas, separadas por comas.
- Manejar frases vacías.
