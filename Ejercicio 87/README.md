# Problema

Plataforma de e-commerce para ofertas de temporada en ropa.

## Restricciones

- Cantidad de camisas ≥ 0.
- Si el total es $0, mostrar 'Compra vacía'.
- Aplicar descuentos en cascada (20% o 10%).
