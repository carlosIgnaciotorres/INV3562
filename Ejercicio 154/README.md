# Problema

Herramienta de ciberseguridad para encriptar mensajes simples.

## Restricciones

- Ignorar números y símbolos en la palabra.
- Si la entrada está vacía, mostrar 'Texto inválido'.
- Conservar mayúsculas/minúsculas al invertir.
