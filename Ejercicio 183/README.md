# Problema

Sistema de validación de tarjetas de crédito para comercios.

## Restricciones

- Número debe tener entre 13 y 19 dígitos.
- Aplicar algoritmo de Luhn para verificar validez.
- Ocultar todos los dígitos excepto los últimos 4.
