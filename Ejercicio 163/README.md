# Problema

Sistema de becas universitarias basado en mérito académico.

## Restricciones

- Promedio ≥ 8.5 y nivel socioeconómico ≤ 3 (escala 1-5).
- Si no cumple, mostrar 'No elegible'.
- Priorizar estudiantes con promedio > 9.0.
