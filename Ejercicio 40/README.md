# Problema

Software para diseño de rutas de drones en tiempo real.

## Restricciones

- Los catetos deben ser mayores que 0.
- Mostrar la hipotenusa con 4 decimales.
- Validar que las unidades ingresadas (metros, pies) sean consistentes.
