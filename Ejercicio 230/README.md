# Problema

App lingüística para detectar anagramas en juegos de palabras.

## Restricciones

- Ignorar espacios y mayúsculas.
- Si son anagramas, mostrar 'Válido' en verde.
- Manejar caracteres especiales (ej: 'í', 'ñ').
