# Problema

Herramienta de logopedia para análisis de pronunciación.

## Restricciones

- Diferenciar mayúsculas y minúsculas (ej: 'A' ≠ 'a').
- Si la palabra contiene números, mostrar 'Error'.
- Mostrar recuento por vocal.
