# Problema

Menú interactivo para cajeros automáticos de bancos.

## Restricciones

- Opciones válidas: 1 (Retiro), 2 (Depósito), 3 (Consulta).
- Si se ingresa una opción inválida, repetir hasta 5 veces.
- Mostrar saldo después de cada transacción.
