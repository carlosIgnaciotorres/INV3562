# Problema

Validador de códigos de producto en almacenes industriales.

## Restricciones

- El número debe estar entre 100 y 1000 (sin incluir 1000).
- Si no es múltiplo de 5, mostrar 'Rechazado' en rojo.
- Ignorar decimales y caracteres no numéricos.
