# Problema

Sistema de reservas para verificar disponibilidad de citas.

## Restricciones

- Día entre 1-31, mes entre 1-12, año ≥ 2024.
- Validar febrero en años bisiestos.
- Si es inválido, mostrar 'Fecha no disponible'.
