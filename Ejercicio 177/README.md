# Problema

Herramienta de empaque para calcular volúmenes de cajas en logística.

## Restricciones

- Dimensiones (largo, ancho, alto) > 0.
- Mostrar resultado en cm³ con 1 decimal.
- Si una dimensión es cero, mostrar 'Error'.
