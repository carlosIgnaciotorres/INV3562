# Problema

Plataforma de estudios demográficos para análisis de género en regiones.

## Restricciones

- Total de personas debe ser ≥ 1.
- Mostrar porcentajes con 1 decimal (ej: 50.0% varones).
- Si hay datos faltantes, mostrar 'Incompleto'.
