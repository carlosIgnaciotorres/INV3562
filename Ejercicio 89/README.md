# Problema

App familiar para gestionar permisos de menores de edad.

## Restricciones

- Edades deben ser ≥ 0.
- Si ambos hermanos tienen la misma edad, mostrar 'Sin diferencia'.
- Resaltar la edad del mayor en verde.
