# Problema

Herramienta para generar series numéricas en análisis financiero.

## Restricciones

- Limitar a 30 múltiplos para evitar carga computacional.
- Si el número es cero, mostrar 'Inválido'.
- Formato: [múltiplo1, múltiplo2, ...].
