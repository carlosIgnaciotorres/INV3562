# Problema

Herramienta de juegos de mesa para generar reglas basadas en paridad.

## Restricciones

- Suma de dígitos debe ser un entero ≥ 0.
- Si la suma es par, mostrar 'Jugador 1 inicia'.
- Ignorar números negativos.
