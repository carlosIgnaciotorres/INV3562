# Problema

Juego de azar para sorteos en eventos promocionales.

## Restricciones

- Generar números entre 1 y 6.
- Mostrar resultado como emoji (ej: ⚀ para 1).
- Validar que el usuario ingrese 'lanzar' para iniciar.
