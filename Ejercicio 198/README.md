# Problema

App interactiva para juegos de azar con reglas basadas en múltiplos.

## Restricciones

- Números enteros positivos.
- Si se ingresa 0, finalizar sin contar.
- Mostrar mensaje: 'Múltiplo de 7 encontrado'.
