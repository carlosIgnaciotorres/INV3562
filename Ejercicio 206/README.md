# Problema

App de edición de texto en tiempo real para corrección automática.

## Restricciones

- Reemplazar solo 'a' por 'e' (case-sensitive).
- Si no hay coincidencias, mostrar el texto original.
- Conservar formato de espacios y saltos de línea.
