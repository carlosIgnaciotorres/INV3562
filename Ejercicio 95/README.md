# Problema

App de análisis de mercado para clasificar tendencias económicas.

## Restricciones

- Números pueden ser positivos, negativos o cero.
- Si es cero, mostrar 'Neutro' en amarillo.
- Usar etiquetas claras: Positivo/Negativo/Neutro.
