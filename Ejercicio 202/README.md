# Problema

Sistema de validación de contraseñas para entornos corporativos.

## Restricciones

- Verificar repeticiones consecutivas (ej: 'aa', 'BB').
- Si no hay repeticiones, mostrar 'Segura'.
- Diferenciar mayúsculas y minúsculas.
