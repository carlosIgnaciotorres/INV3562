# Problema

Simulador de movilidad urbana para vehículos autónomos.

## Restricciones

- Velocidad inicial y final deben ser ≥ 0.
- Tiempo debe ser en segundos (valores enteros).
- Mostrar espacio en metros con 3 decimales.
