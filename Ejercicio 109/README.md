# Problema

App de calendario para gestión de eventos históricos.

## Restricciones

- Año debe ser ≥ 1582 (calendario gregoriano).
- Si es divisible por 400, es bisiesto.
- Mostrar 'Bisiesto' o 'No bisiesto'.
