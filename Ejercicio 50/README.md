# Problema

Sistema de logística para agrupar paquetes en pallets.

## Restricciones

- Solo números enteros positivos.
- Si es múltiplo de 3, mostrar 'Agrupable'. Si no, 'Rechazado'.
- Validar que la entrada no sea cero.
