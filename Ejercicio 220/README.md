# Problema

Herramienta de redes sociales para ordenar seguidores alfabéticamente.

## Restricciones

- Ignorar mayúsculas/minúsculas al ordenar.
- Si hay nombres duplicados, mostrar solo uno.
- Limitar a 100 nombres.
