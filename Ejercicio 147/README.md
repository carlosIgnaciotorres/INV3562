# Problema

Herramienta de criptografía para validar palíndromos numéricos.

## Restricciones

- Números deben tener al menos 2 dígitos.
- Ignorar signos negativos (ej: -121 → 121).
- Mostrar 'Capicúa' o 'No capicúa'.
