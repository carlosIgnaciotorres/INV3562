# Problema

Plataforma de estudios demográficos para análisis de género.

## Restricciones

- Total de participantes debe ser ≥ 1.
- Mostrar porcentajes con 1 decimal (ej: Hombres: 55.5%).
- Si los datos están desbalanceados (>90%), mostrar advertencia.
