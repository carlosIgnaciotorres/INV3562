# Problema

Sistema de detección de plagio en textos académicos.

## Restricciones

- Ignorar mayúsculas/minúsculas y caracteres especiales.
- Si el texto está vacío, mostrar 'Entrada inválida'.
- Validar palíndromos de al menos 3 caracteres.
