# Problema

App de gestión de recursos hídricos para calcular tiempos de llenado de tanques.

## Restricciones

- Capacidad del tanque: 50 m³ (fija).
- Caudal en litros/minuto debe ser > 0.
- Mostrar horas con 1 decimal (ej: 12.5 horas).
